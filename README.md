# simple-framework

简易框架

## Install

```bash
$ npm install git://github.com:kiba-zhao/egg-simple-framework.git --save
```

## Configuration

```js
// {app_root}/package.json
{
  "egg": {
    "framework": "simple-framework"
  }
}
```
